Example 1

This is a default version. The star comes from FontAwesome and it works as a fallback.

<input-stars ng-model="prop1" on-star-click="clickHandler(prop1)" max="5"></input-stars>

Example 2

You can change the icon when is empty with icon-empty and when is full with icon-full

value: {{prop2}} 
<input-stars icon-empty="fa-heart-o" icon-full="fa-heart" ng-model="prop2" max="5"></input-stars>
<code>

Example 3

You can bootstrap icons too

value: {{prop3}} 
<input-stars icon-base="glyphicon" icon-empty="glyphicon-remove" icon-full="glyphicon-ok" ng-model="prop3" max="5"></input-stars>
<code>

Example 4

You can choose if you want to use different icons when the user sets a rating

value: {{prop4}} 
<input-stars icon-empty="fa-heart-o" icon-full="fa-twitter" ng-model="prop3" max="5"></input-stars>
<code>

Example 5

Set the maximum rating with an scope variable

value: {{prop5}} 
<input-stars ng-model="prop4" max="{{maxDynamic}}"></input-stars>
<code>

Example 6

You can use Material Icons that uses ligature for its icons only with the attribute ligature

value: {{prop6}} 
<input-stars icon-base="material-icons" icon-empty="sentiment_dissatisfied" icon-full="sentiment_satisfied" ng-model="prop5" max="5" ligature></input-stars>
<code>

Example 7

You can make an input star readonly easily, just can simple add the readonly attribute

value: {{prop7}} 
<input-stars ng-model="prop1" max="5" readonly></input-stars>
<code>

Example 8

You play with a variable to chance the icons when the input is readonly

Toggle readonly {{ enableReadonly }}
value: {{prop2}} 
<input-stars ng-model="prop1" max="5" ng-attr-readonly="{{ enableReadonly || false }}" ng-attr-icon-empty="{{ enableReadonly ? 'fa-twitter' : 'fa-circle-o' }}" ng-attr-icon-full="{{ enableReadonly ? 'fa-cog' : 'fa-twitter' }}"></input-stars>
<code>

Example 9

Change the class on hover using the attribute icon-hover="classname"

red blue {{ hoverClass }}
value: {{prop2}} 
<input-stars ng-model="prop2" max="5" icon-hover="{{hoverClass}}"></input-stars>
<code>

Example 10

Adding a click event to the element it's easy, just add on-star-click="functionHandler()" and that's it. (Check the console)

value: {{prop3}} 
<input-stars ng-model="prop3" max="5" on-star-click="clickEventHandler()"></input-stars>
